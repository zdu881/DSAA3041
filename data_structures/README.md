# 数据结构 (Data Structures)

本目录包含各种数据结构的实现和练习。

## 目录结构

- **linear/**: 线性数据结构
  - 数组、链表、栈、队列、双端队列等
  
- **tree/**: 树形数据结构
  - 二叉树、二叉搜索树、AVL树、红黑树、堆、字典树等
  
- **graph/**: 图数据结构
  - 邻接表、邻接矩阵、图的遍历算法等
  
- **hash/**: 哈希数据结构
  - 哈希表、哈希集合、一致性哈希等

## 使用方法

1. 选择合适的子目录
2. 创建新文件或使用模板
3. 实现数据结构及其操作
4. 添加测试用例验证正确性
