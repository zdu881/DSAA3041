# 代码模板 (Templates)

本目录包含各种编程语言的问题解决模板。

## 可用模板

- **problem_template.cpp**: C++ 问题解决模板
- **problem_template.py**: Python 问题解决模板
- **problem_template.java**: Java 问题解决模板

## 使用方法

1. 复制相应语言的模板文件
2. 重命名为你的问题名称
3. 填写问题描述、来源、难度、标签等信息
4. 实现解决方案
5. 添加测试用例

## 模板内容

每个模板包含：
- 问题基本信息（名称、来源、难度、标签）
- 问题描述区域
- 解题思路说明
- 时间和空间复杂度分析
- 解决方案框架
- 测试用例框架
