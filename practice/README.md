# 练习题 (Practice Problems)

本目录包含来自各个平台的练习题及其解答。

## 目录结构

- **leetcode/**: LeetCode 平台的题目
- **codeforces/**: Codeforces 平台的题目
- **other/**: 其他平台或自定义题目

## 使用方法

1. 选择合适的平台目录
2. 为每道题创建单独的文件或目录
3. 使用 `/templates` 目录下的模板开始解题
4. 在文件头部标注题目信息、难度、标签等
5. 记录解题思路和复杂度分析

## 命名规范

建议使用以下命名格式：
- `problem_id_problem_name.ext`
- 例如：`001_two_sum.py`, `102_binary_tree_level_order_traversal.cpp`
