# 算法 (Algorithms)

本目录包含各种算法的实现和练习。

## 目录结构

- **sorting/**: 排序算法
  - 冒泡排序、选择排序、插入排序、快速排序、归并排序、堆排序等
  
- **searching/**: 搜索算法
  - 线性搜索、二分搜索、深度优先搜索(DFS)、广度优先搜索(BFS)等
  
- **dynamic_programming/**: 动态规划
  - 背包问题、最长公共子序列、最短路径等
  
- **greedy/**: 贪心算法
  - 活动选择、哈夫曼编码等
  
- **divide_conquer/**: 分治算法
  - 归并排序、快速排序、大整数乘法等

## 使用方法

1. 选择合适的子目录
2. 创建新文件或使用模板
3. 实现算法
4. 添加测试用例验证正确性和性能
